# Project Title

Library for showing numpy images on jupyter notebook.

### Prerequisites

* [matplotlib](https://matplotlib.org/)
* [numpy](https://numpy.org/)

### Usage
```
import imshowlib as imshow
```
To use imshow3d, after import do:
```
%matplotlib inline
```
or
```
%matplotlib notebook
```
